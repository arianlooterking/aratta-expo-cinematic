# Aratta Expo Cinematic

Standalone Next.js redesign for Aratta Expo / شرکت توسعه تجارت اَرَت.

## Commands

```bash
npm install
npm run dev -- --hostname 127.0.0.1 --port 9090
npm run lint
npm run typecheck
npm test
npm run build
```

## Product Boundaries

- Persian is primary (`/fa`) and English has parity (`/en`).
- Current public exhibition dates are before May 26, 2026, so events are archived until a fresh calendar is supplied.
- Registration is an inquiry/download flow. The form creates an email draft and does not fake backend submission.
- Official forms/downloads remain linked to `arattaexpo.ir`.

## Implemented Surface

- Scroll-linked cinematic booth-build hero with reduced-motion fallback.
- Liquid-glass nav, language toggle, mobile drawer, exhibition archive, services tabs, official downloads, inquiry form, gallery lightbox, news archive, contact section, sitemap, robots, and JSON-LD.
- Local QA screenshots are in `qa/` when generated by verification scripts.

## Known Launch Note

`npm audit --audit-level=moderate` currently reports a moderate advisory through Next.js internal `postcss`; npm only offers `npm audit fix --force`, which would downgrade Next to an old breaking version. Do not run the forced fix blindly.
